# Memory Ball Game

**Author:** Chao Li
**Date:** July 25, 2025  

A browser-based memory game built with **HTML, CSS, and JavaScript**. The game challenges the player to remember the order of numbered balls and click them correctly. It demonstrates SVG manipulation, animations, and front-end interactivity.

---

## Features

- **Dynamic sequence generation** – Balls are displayed in a random non-overlapping pattern each level.  
- **Memory challenge** – Numbers appear briefly and then hide, requiring the player to recall the order.  
- **Level progression** – Successfully completing a sequence increases the level and sequence length.  
- **Attempts tracking** – Player has 3 attempts displayed as heart icons.  
- **Timer with progress bar** – 3-minute countdown with color-changing bar.  
- **Visual feedback** – Correct clicks flash green, incorrect clicks shake red.  
- **High score persistence** – Highest level reached is saved using `localStorage`.  
- **Responsive design** – Works on desktop and tablet screens.  
- **Interactive SVG** – Hover effects and click animations make gameplay engaging.

---

## How to Play

1. Click **Start Game**.  
2. Watch the numbered balls appear in sequence.  
3. Numbers disappear.  
4. Click the balls in the correct order.  
5. Correct selections turn green, incorrect selections turn red.  
6. Complete the sequence to advance to the next level.  
7. You have 3 attempts and 3 minutes to reach the highest level.

---

## Technology Stack

- **HTML5** – Structure and layout  
- **CSS3** – Styling, animations, hover and click effects  
- **JavaScript (ES6)** – Game logic, event handling, timer, localStorage  
- **SVG** – Balls and animations  
- **Bootstrap 5** – Responsive layout
